# Ultra image webpage

## Setup

### Prerequisite
````bash
gem install bundler

# Optional, since this should be installed with bundler in the next step.
gem install jekyll-compose
````

### Install
````bash
bundle install
````

### Update
````bash
bundle update
# OR
bundle update github-pages
````

### Run (Requires: NodeJS)
````bash
npm install
npm start
````
